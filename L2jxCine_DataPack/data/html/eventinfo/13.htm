Mutant
Event Information
1 random player becomes a mutant
Mutant gets: 3x p.def/m.def 2x p.atk/m.atk/c.spd/atk.spd and 30% resist to debuffs
Only the mutant can score
Everyone else can kill mutant only
If 2 or more players get same score then a random player wins
Event reward: 2 apigas
Event duration: 4 minutes
Levels allowed: 70-80
Minimum players: 8
Self buff event
Ressurection time: 7 Seconds
Rules
Be Warned!
Dualboxing is not allowed during event
Feeding enemy team is not allowed, this means buffing/going in naked etc.
In general if a char registers just to ruin event he gets risked getting punished.
