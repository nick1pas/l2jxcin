Last man standing
Event Information
Free for all event
The last person to stay alive is the winner
Reward for every 2 kills: 1 Medal
Event reward: 25 medals
Levels allowed: 70-80
Event duration: 5 minutes
Minimum players: 8
Full buff event
Ressurection time: None
Rules
Be Warned!
Dualboxing is not allowed during event
Feeding enemy team is not allowed, this means buffing/going in naked etc.
In general if a char registers just to ruin event he gets risked getting punished.
